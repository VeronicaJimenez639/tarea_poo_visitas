# tarea_poo_visitas

Aplicación desarrollada en Python con Tkinter para gestionar el registro de visitantes.

## Funcionalidades
- Registrar visitantes
- Visualizar visitantes en una tabla
- Actualizar registros seleccionados
- Eliminar registros
- Limpiar campos del formulario

## Estructura
- `modelos/`: contiene la clase `Visitante`
- `servicios/`: contiene la lógica CRUD
- `ui/`: contiene la interfaz gráfica en Tkinter
- `main.py`: punto de entrada de la aplicación

## Cómo ejecutar
1. Abrir el proyecto en PyCharm o en el editor de preferencia.
2. Verificar que Python esté instalado.
3. Ejecutar el archivo `main.py`.

## Requisitos
- Python 3.x
- Librería estándar `tkinter`
